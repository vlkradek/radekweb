tools, which I use

HTML

HTML, in full hypertext markup language, a formatting system for displaying material retrieved over the Internet.

css

Cascading Style Sheets. An extension to HTML which are most often used to set default styles so that the equivalent HTML code does not need to be retyped.

JAVASCRIPT

A powerful, object-based, interpreted scripting language, most commonly embedded directly into HTML web pages to manage client-side interaction.

Sass

A CSS preprocessor is a scripting language that extends CSS by allowing developers to write code in one language and then compile it into CSS.

Figma

Figma is a web-based graphics editing and user interface design app. You can use it to do all kinds of graphic design work from wireframing websites, designing mobile app interfaces, and much more
